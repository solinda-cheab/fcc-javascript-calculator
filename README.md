#  JavaScript Calculator

A modern JavaScript Calculator built with React for the FreeCodeCamp Front End Development Libraries Certification. This calculator supports basic arithmetic operations, decimal calculations, operator precedence, and provides a responsive, user-friendly interface.

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/fcc-javascript-calculator.git
```

Navigate to the project directory:

```bash
cd fcc-javascript-calculator
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
